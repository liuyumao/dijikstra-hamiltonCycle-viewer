##### 3. 不足之处

1. 程序界面主题色彩比较单一，二次开发者应该再琢磨琢磨QSS样式对控件美化。
2. 程序的健壮性不强，当存在非法输入时不能完全检测出，程序接收非法数据时易崩溃。
3. 程序界面的QGraphicsView中所画的点与线可以再做优化，特别是在颜色和形状方面，对于本程序所画的点，为一个圆形，填充的颜色为黄色，点与点所连线段为黑色。当给出起点和终点时，如果存在一条最短路径，那么，这条路径上的边会变成红色。
